# site-fifa2026-guide-ad0f

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的静态仓库。仓库内包含一系列与 2026 年 Fédération Internationale de Football Association 相关主题的指南类页面，以 HTML 格式存储，方便直接查看或部署。

所有页面均为独立文件，不依赖外部数据库或后端服务，仅需静态托管即可运行。

## 目录说明

```
.
├── index.html          # 入口页面（如有）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：核心目录，存放所有独立 HTML 页面，每个页面可独立访问。
- **assets/**：存放页面共用的样式、脚本或图片资源。
- **index.html**：可选的入口页面，用于导航或跳转。

## 页面归档说明

- 每个 HTML 页面均为自包含文件，包含完整的页面结构与样式。
- 页面之间无硬依赖，可单独部署或组合使用。
- 所有页面均以通用、简洁的布局呈现，便于快速理解内容。
- 仓库内不包含任何动态生成或实时数据接口，所有内容已静态化。

## 维护说明

- 新增页面：在 `pages/` 目录下创建新的 `.html` 文件，并确保资源路径正确。
- 更新页面：直接修改对应 HTML 文件，无需额外配置。
- 资源管理：如需添加公共资源，请放入 `assets/` 对应子目录，并在页面中通过相对路径引用。
- 本仓库以归档为主，不提供在线预览或 CI/CD 自动部署配置，使用者可自行托管至任何静态站点服务。
